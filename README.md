# Streaks

A simple, beautiful habit tracker that helps you build consistency one day at a time.

**Live demo:** [streaks-bitcoineo.vercel.app](https://streaks-bitcoineo.vercel.app)

![Screenshot](screenshot.png)

## Tech Stack

- **Next.js 14** — App Router, TypeScript, server components
- **Tailwind CSS** — Utility-first styling
- **Drizzle ORM** — Type-safe database queries
- **Turso / libSQL** — SQLite database (edge-compatible)
- **Auth.js v5** — JWT sessions, email/password + Google OAuth
- **Zod** — Schema validation
- **Vercel** — Deployment

## Features

- Email/password and Google OAuth sign-in
- Create habits with name, emoji, and frequency (daily or weekly)
- One-tap completion toggle with optimistic UI
- Streak tracking and 7-day completion rates
- Week view grid showing daily progress
- Dashboard stats with progress bars
- Settings page for profile management
- Fully isolated per-user data
